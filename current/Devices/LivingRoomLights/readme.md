# Living Room Lights

This is the 4 ceiling strips in the living room.

Running on a ESP32 dev board

## Pins

### Inputs

On/Off switch - Input pullup: D2, Pin4, GPIO2 

### Outputs

Strip 1: 12
Strip 2: 14
Strip 3: 27
Strip 4: 26
